# Web Aero

A faithful recreation of Windows 7 in HTML, CSS, and TypeScript

## Development

### Prerequisites

| Resource | Version | Download                                              |
| -------- | ------- | ----------------------------------------------------- |
| Node.js  | 18.17.0 | [https://nodejs.org](https://nodejs.org/en/download/) |
| Yarn     | 1.22.10 | `npm i -g yarn`                                       |

### Setup

```shell
> yarn
> yarn dev
```
